# Homework  UDP Programming

## 1. Сонсольний чат в локальній мережі

Створити чат в локальній мережі, який буде працювати на протоколі UDP. Чат має підтримувати наступні функції:
- відправлення повідомлення всім користувачам чату
- відправлення приватного повідомлення конкретному користувачу
- блокування користувача

<b>PS. Для ідентифікації користувача в чаті використовувати IP адресу. При вході в чат користувач вводить свій нікнейм і відправляє його по мережі brodcast. </b>

Для демонстрації і перевірки працезданості чату потрібні як мінімум два комп'ютери.
Рекомендую використовувати віртуальні машини з встановленим [ubuntu 22.04 LTS](https://ubuntu.com/download/server/thank-you?version=22.04.5&architecture=amd64&lts=true).

Мінімальні вимоги до WM - 1 ядро, 1 Гб оперативної пам'яті, 10 Гб дискового простору. (тимчасово для інсталяції збільшуєте оперативну память до 2Gb або встановлюєте динамічне виділення памяті )
