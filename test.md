# Test

Написати програму в WPF яка буде графічно відображати прогрес розрахунку матриць за допомогою Thread,Task в прогрес барах.

Передбачити можливість:
- вибору розмірності матриці
- вибору прогрес бару який прийме участь в тесті (чекбокс напроти гріда)
- припинення роботи теста (це повне передчасне завершення тесту по кожному завданню
- опціонально додати для Thread вібір приорітету.
